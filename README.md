# Ansible Role: pimoroni-ledshim

An Ansible Role that installs [Pimoroni LEDShim](https://shop.pimoroni.com/products/led-shim?variant=3136952467466) on Raspberry Pi.

## Requirements

None.
## Role Variables

None.

## Dependencies

None.

## Example Playbook

Include in the playbook:

```bash
      roles:
         - sund.pimoroni_ledshim
```
